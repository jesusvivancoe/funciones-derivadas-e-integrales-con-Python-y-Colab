# Cálculo Interactivo con Python y Colab

Este repositorio contiene **notebooks interactivos** para aprender y practicar cálculo de manera visual y práctica usando Python en Google Colab.  
Se incluyen ejemplos de:

- Funciones y sus gráficas.
- Derivadas y su interpretación gráfica.
- Integrales definidas e indefinidas con visualización de áreas.
- Aplicaciones de cálculo a problemas reales y simulados.

Los notebooks combinan **teoría en notación matemática (LaTeX)**, **código ejecutable** y **gráficos interactivos**, para que los estudiantes puedan explorar los conceptos de manera dinámica.
